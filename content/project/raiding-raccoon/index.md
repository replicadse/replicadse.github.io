## Why

Well... I created this because I was interested in the network that links create on a website. Checking the fastest path from one page to another.\
It turned out that a proper graph representation of pages that are linked to another can do really cool things. Especially when using the exported graphml format and loading it into an application like [Gephi](https://gephi.org) and then using the [ForceAtlas 2](https://arxiv.org/pdf/1905.02202.pdf) simulation. This creates really interesting insights.
